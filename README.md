# Cyberia Insurance – P&C DCF Valuation Model (v1)

Hypothetical property & casualty insurer valuation  
6-year explicit forecast with full claims triangle, reserve runoff, and float modeling.

### Key Assumptions
- Base premiums: $100 million
- Premium growth: 5% → terminal 2%
- Loss ratio: 70% | Expense ratio: 25% → Combined ratio 95%
- Investment yield on float: 4%
- WACC: 11.4216%

### Current Equity Value
**$404.9 million** (base case)
